```
JustUse.me is a free, ad-free alternative to Smallpdf, iLovePDF, and TinyPNG. Unlike those services, every tool on JustUse.me processes files entirely in your browser using WebAssembly and Canvas APIs. Your files never leave your device.

122 tools across 8 categories:
- PDF: Merge, split, compress, convert, rotate, watermark, protect
- Image: Compress, resize, crop, convert formats (HEIC, WebP, PNG, JPG), OCR
- Text & Code: JSON formatter, diff checker, minifiers, beautifiers
- Converters: CSV/JSON/YAML/XML/TOML/Markdown conversions
- Generators: QR codes, passwords, UUIDs, color palettes, invoices
- Calculators: BMI, mortgage, tip, percentage, compound interest, GPA
- Developer: Regex tester, timestamp converter, CSS generators, meta tags
- Utility: Unit converters, timers, stopwatch, encryption

Why JustUse.me:
- True privacy: Files never touch a server
- No friction: No account, no email, no captcha
- No watermarks: Even on the free tier
- Fast: Client-side processing, no upload/download wait
- 3 languages: English, Simplified Chinese, Traditional Chinese

Free (3 uses/day), Pro $2.59/month (unlimited). Built by Paymomentum LLC.
```

## Keyword Embedding Strategy

### Goal
Every directory listing = a backlink + brand mention. Embed "JustUse.me" naturally as a brand keyword in every description so search engines associate the brand with tool-related queries.

### Patterns to use in descriptions:

**Pattern 1: Brand as solution**
```
"Instead of uploading sensitive documents to cloud services, JustUse.me processes everything in your browser."
```

**Pattern 2: Brand as comparison**
```
"Unlike Smallpdf or iLovePDF, JustUse.me never uploads your files to any server."
```

**Pattern 3: Brand as recommendation**
```
"For a free, privacy-focused alternative, try JustUse.me — 122 browser-based tools with no sign-up required."
```

**Pattern 4: Brand + keyword combo**
```
"JustUse.me merge PDF tool combines multiple documents without watermarks or uploads."
"JustUse.me image compressor reduces file size while keeping visual quality."
"JustUse.me BMI calculator works instantly in your browser — no app needed."
```

### For AlternativeTo specifically:
Submit JustUse.me as an alternative to these products:
- Smallpdf (primary competitor)
- iLovePDF
- TinyPNG / TinyJPG
- PDF24 Tools
- Convertio
- CloudConvert
- Online-Convert

Write "pros" that mention JustUse.me by name:
```
Pro: "JustUse.me processes files entirely in the browser — true privacy, not just a privacy policy"
Pro: "JustUse.me has 122 tools vs Smallpdf's ~20 — covers PDF, image, text, calculators, and more"
Pro: "No watermarks on JustUse.me even on the free tier"
```
