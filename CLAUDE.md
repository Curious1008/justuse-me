### Quick commands

```bash
# Verify all credentials
python3.13 ~/.claude/skills/seo/scripts/google_auth.py --check

# List GSC properties
python3.13 ~/.claude/skills/seo/scripts/gsc_query.py sites

# Top queries (28d)
python3.13 ~/.claude/skills/seo/scripts/gsc_query.py query \
  --property "sc-domain:justuse.me" --days 28 --dimensions query --limit 50 --json

# Top pages
python3.13 ~/.claude/skills/seo/scripts/gsc_query.py query \
  --property "sc-domain:justuse.me" --days 28 --dimensions page --limit 50 --json

# URL inspection (indexation status)
python3.13 ~/.claude/skills/seo/scripts/gsc_inspect.py \
  --site "sc-domain:justuse.me" --url "https://www.justuse.me/<path>"

# PageSpeed / CrUX
python3.13 ~/.claude/skills/seo/scripts/pagespeed_check.py --url https://www.justuse.me/
python3.13 ~/.claude/skills/seo/scripts/crux_history.py --url https://www.justuse.me/
```

## Project layout

- Frontend: Next.js 16 App Router, `src/app/[lang]/` (en, zh-CN, zh-TW)
- Tool plugin system: `src/tools/<category>/<id>.ts` implementing `ToolPlugin`
- SEO articles: ISR via `src/app/[lang]/news/`, generated by `~/justuseme-cron/` (Cloudflare Worker + Claude API)
- Auth: Supabase. Payments: Stripe. Deploy: Vercel.
